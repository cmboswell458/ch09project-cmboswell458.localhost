# CH09 WordPress Project

## Project Overview

This project demonstrates the installation, configuration, and customization of a WordPress website using AMPPS, MySQL, GitHub, and Visual Studio Code. The website was developed as part of a web development assignment focused on WordPress content management systems and version control integration.

## Features

* WordPress installation using AMPPS
* Custom localhost domain configuration
* MySQL database setup with phpMyAdmin
* Git and GitHub repository integration
* Custom Astra WordPress theme activation
* Static homepage configuration
* Navigation menu creation
* Widget implementation
* Plugin installation and activation
* Customized site title, tagline, and favicon
* Block editor content creation
* Responsive design customization
* WordPress Site Health review

## Technologies Used

* WordPress
* AMPPS
* PHP
* MySQL
* Git
* GitHub
* Visual Studio Code

## Project Structure
